## expression

- In Python, an **expression** is a **combination of**  
  - values,
  - variables,
  - operators, and
  - function calls  
  
  … that **can be evaluated** to **produce a single value**.

## statement

- see [statement](https://realpython.com/ref/glossary/statement/)

- In Python, a **statement** is a **complete instruction**  
  … that tells the Python interpreter to **perform an action**.

- *Statements* **execute operations** or **control program flow**.

- The *key distinction* between *statements* and [*expressions*](#expression) is  
  … that *statements* **perform actions**  
  … rather than just *evaluating to values*.
